# Shared Design References

This directory contains static HTML design artifacts.

- `teamwork-design.html` is the original Teamwork design proposal. It is useful for design intent, but the current implementation has superseded some details.
- `teamwork-ui-mockup.html` is a static mockup that reuses production CSS for layout reference.

The current source of truth is the implementation in `src/`, the tests in `tests/`, and the Markdown docs in `docs/`.
